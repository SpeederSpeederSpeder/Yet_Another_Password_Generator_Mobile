# Yet_Another_Password_Generator
Just generate a random password, right?

# Features
Multilanguage support

Customizable password length with a slider

Customizable password characters

Generated passwords history

Password strength indicator

Dark and light switch

# How to host on your computer
1. Clone the repository
    ```bash
    git clone
2. Install dependencies
    ```bash
    npm install
3. Start the development server
    ```bash
    npm run dev
4. Open your browser and go to `http://localhost:5173`
5. Enjoy the app!

# How to host with Vercel
1. Go to [Vercel](https://vercel.com/)
2. Sign up or log in
3. Import the repository
4. Follow the instructions to deploy
5. Once deployed, you will get a live URL to access the app
