# Supported Languages

This document lists the languages supported by the application. Each language is represented by a JavaScript file in the `languages/` directory.

* Arabic (ar.js)
* Czech (cs.js)
* Danish (da.js)
* German (de.js)
* English (en.js)
* Spanish (es.js)
* Finnish (fi.js)
* French (fr.js)
* Hebrew (he.js)
* Hindi (hi.js)
* Indonesian (id.js)
* Italian (it.js)
* Japanese (ja.js)
* Korean (ko.js)
* Malay (ms.js)
* Dutch (nl.js)
* Norwegian (no.js)
* Polish (pl.js)
* Portuguese (pt.js)
* Romanian (ro.js)
* Russian (ru.js)
* Swedish (sv.js)
* Thai (th.js)
* Turkish (tr.js)
* Ukrainian (uk.js)
* Vietnamese (vi.js)
* Chinese (zh.js)
